# Audio STT (Speech-to-Text)

Convert audio to text using OpenAI Whisper or local STT engines.

## Purpose
Transcribe audio files into text with high accuracy and multi-language support.

## Supported Features
- Multiple audio formats (MP3, WAV, M4A, etc.)
- Multi-language recognition
- Timestamped transcriptions
- Local and cloud-based processing

## Input
Audio file path or URL

## Output
Transcribed text with optional timestamps
