# OpenAI Whisper API

Cloud-based speech-to-text using OpenAI Whisper API.

## Purpose
Transcribe audio to text using OpenAI's Whisper API with high accuracy and fast processing.

## Supported Features
- Cloud-based transcription (requires internet)
- Multi-language support
- High accuracy
- Fast processing
- Support for various audio formats
- Timestamped transcriptions

## Setup
Requires OpenAI API key (OPENAI_API_KEY)

## Input
Audio file or URL

## Output
Transcribed text with optional timestamps and language detection
