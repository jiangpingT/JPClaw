# Audio TTS (Text-to-Speech)

Convert text to natural-sounding speech using OpenAI TTS or local TTS engines.

## Purpose
Generate high-quality speech audio from text input.

## Supported Features
- Multiple voices and styles
- Multi-language support (Chinese, English, etc.)
- Cloud (OpenAI TTS) and local (Sherpa ONNX) options
- Custom speed and pitch control

## Input
Text content to be spoken

## Output
Audio file (MP3, WAV, etc.)
