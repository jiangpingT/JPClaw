# Weather Skill

查询实时天气，输出个性化播报。

## 播报规则

**北京（家）：**
1. 今日温度 + 与昨日对比（降幅 ≥ 5°C 提醒孩子加衣）
2. 是否备伞（降雨概率 ≥ 40% 提醒）
3. 极端天气（有则标出）
4. 穿衣建议（大宝约15岁 / 二宝约7岁）

**外地（出差）：**
1. 极端天气
2. 穿衣建议

## Input

JSON 字段：
- `city`（可选）：城市名，默认「北京」
- `location`（可选）：同 city

## Output

JSON：
- `ok`：是否成功
- `city`：查询城市
- `isHome`：是否为北京
- `report`：天气播报文本
