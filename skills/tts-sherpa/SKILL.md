# Sherpa ONNX TTS

Local text-to-speech synthesis using Sherpa ONNX engine (offline, privacy-friendly).

## Purpose
Generate speech audio locally without cloud API calls, using ONNX runtime.

## Supported Features
- Offline TTS (no internet required)
- Multi-language support
- Multiple voice models
- Fast local processing
- Privacy-friendly (no data sent to cloud)

## Setup
Requires Sherpa ONNX models downloaded locally

## Input
Text to be spoken

## Output
Audio file (WAV, MP3, etc.)
