# Obsidian

Manage Obsidian markdown notes in your local vault.

## Purpose
Create, search, edit, and organize notes in Obsidian vaults.

## Supported Features
- Create new notes with templates
- Search notes by content or tags
- Edit existing notes
- Manage bi-directional links
- Tag management
- File operations (rename, move, delete)

## Setup
Requires path to Obsidian vault directory

## Input
Note content, search queries, or file paths

## Output
Created/updated note paths or search results
