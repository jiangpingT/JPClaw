# Speak Skill

通过 macOS `say` 命令用普通话语音（婷婷）播报文字，声音通过电脑扬声器输出。

## Input

JSON 字段（均可选）：
- `text`：要播报的文字，默认"请住手，我马上报警了"

## Output

```json
{ "ok": true, "text": "请住手，我马上报警了", "voice": "Tingting" }
```

## 依赖

- macOS 内置 `say` 命令
- 需安装"婷婷"语音包（系统偏好设置 → 辅助功能 → 朗读内容）
