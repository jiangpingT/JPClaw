# Skills (minimal)

Each skill is a folder under `skills/` with a `SKILL.md` definition.

Example structure:
```
skills/echo/
  SKILL.md
```

`SKILL.md`:
```
---
name: echo
description: Echo input text
---

# Echo

## Purpose
Return the input verbatim.

## Inputs
Plain text string.

## Output
Echoed text in the form: `echo: <input>`.
```
